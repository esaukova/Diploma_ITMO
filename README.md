# RED Monitor

Корпоративная система мониторинга сотрудников для гибридного формата работы.

Проект разработан в рамках дипломной работы и представляет собой веб-приложение для:

- отметки формата работы сотрудников;
- мониторинга присутствия;
- ведения истории активности;
- авторизации через LDAP;
- административного контроля.

---

# Возможности системы

## Для сотрудников

- Авторизация через LDAP
- Просмотр календаря
- Отметка статуса работы:
  - В офисе
  - Удалённо
  - Выходной
  - Больничный
- Просмотр собственной активности

---

## Для администратора

- Панель мониторинга
- Статистика сотрудников
- Графики активности
- Просмотр всех записей
- Фильтрация сотрудников по статусам
- Просмотр истории активности

# Авторизация

Система поддерживает:

- LDAP авторизацию
- локальное резервное хранение пользователей

При недоступности LDAP возможна авторизация через локальную базу данных.

---

# База данных

При первом запуске:

- автоматически создаётся база данных Firebird;
- автоматически создаются таблицы;
- автоматически создаются связи между сущностями.

---

# Основные сущности БД

## users

Хранит информацию о пользователях:

- username
- role
- created_at
- last_login
- password_hash

---

## work_logs

Хранит отметки сотрудников:

- user_id
- work_date
- status
